# Agrinho_2026
O EcoAgro 2026 – Raízes do Futuro é uma plataforma web educativa desenvolvida para promover a conscientização ambiental e incentivar práticas sustentáveis no meio rural e escolar.

O projeto foi criado para o Programa Agrinho 2026 e busca demonstrar como a tecnologia pode ser utilizada para disseminar conhecimento sobre sustentabilidade, ESG (Ambiental, Social e Governança), preservação dos recursos naturais e agricultura sustentável.

Através de ferramentas interativas e conteúdos educativos, o site estimula a adoção de atitudes responsáveis que contribuem para a construção de um futuro mais sustentável.
